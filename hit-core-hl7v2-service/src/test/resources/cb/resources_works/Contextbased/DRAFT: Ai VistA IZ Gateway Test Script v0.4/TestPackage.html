1.1.1. Provider Sends VXU for HX 18YO HIB

Test Step Type: TA_INITIATOR
Demonstrates the ability of the provider system to generate a VXU Test Message for an adult 18 year-old veteran being vaccinated with Haemophilus influenzae type b as a single dose formatted according to the IZ Gateway HL7 v2.5.1 IG for Multi-Jurisdiction Provider IZ Messaging.

Test Story

Description

This test evaluates the capability of an Multi-Jurisdictional Provider (Sender) technology to send a SubmitSingleMessage containing a VXU message via the IZ Gateway operation that conforms to the SOAP 1.2 standard and IZ Gateway WSDL. This is verified interactively with the Jurisidiction (IIS) Test Partner which should send the ACK message in response to the VXU message.

An 18-year old veteran is vaccinated with Haemophilus influenzae type b as a single dose.


Comments

Destination Application Identifier (MSH-5) and Destination Facility Identifier (MSH-6) information will change for each IIS partner tested to the IIS self-identified identifier for the destination IIS Application ID and the destination IIS Facility ID respectively. 

Source Facility Identifier (MSH-4) information will change according to the facility identifier issued by the IIS to the IZ Gateway for testing.

Entering Organization (ORC-17) and Administered-at-location (RXA-11) will change according to the facility that is anticipated will be tested as the information source.


Pre-condition

The IIS partner is ready to receive communications from the Multi-Jurisdictional Provider via the IZ Gateway.

The provider system is able to generate a VXU according to the IZ Gateway HL7 v2.5.1 IG for Multi-Jurisdiction Provider IZ Messaging.


Post-Condition

The IIS has successfully sent a VXU message with no critical errors.


Test Objectives

Verify readiness of the IIS partner to receive and respond to IZ-Gateway standard messages: The IIS that will be receiving vaccination information or queries from a multi-jurisdictional provider is configured to successfully receive HL7 messages that conform to the IZ Gateway HL7 v2.5.1 IG for Multi-Jurisdiction Provider IZ Messaging and respond to these messages with associated ACK and RSP HL7 messages that conform to the HL7 2.5.1 Immunization Implementation Guide.

Verify readiness of the Multi-Jurisdictional Provider partner to send standard messages via the IZ-Gateway: The Multi-Jurisdictional Provider that will be sending vaccination information or queries from a is configured to successfully receive send HL7 messages that conform to the IZ Gateway HL7 v2.5.1 IG for Multi-Jurisdiction Provider IZ Messaging.


Evaluation Criteria

Instructions                                                                                                                

  1. The IIS receives the test message defined in the Test Data Specification and Message Content supplied in this test case via the IZ Gateway.
  2. The Acknowledgement message returned from this message is validated in the next test step
